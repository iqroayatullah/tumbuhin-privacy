📍 Tentang Halaman Ini

Aplikasi Tumbuhin (dikembangkan oleh Muhammad Iqro' Ayatullah) menyimpan seluruh data pengguna secara lokal di perangkat masing-masing. Karena tidak ada server pusat yang menyimpan data pengguna, penghapusan data dilakukan langsung melalui perangkat pengguna sendiri, seperti dijelaskan di bawah ini.

🗑️ Cara Menghapus Data Anda

Pilih salah satu metode berikut sesuai kebutuhan Anda:

Metode 1 — Hapus Seluruh Data (Uninstall Aplikasi)

1 Buka pengaturan perangkat Android Anda, atau tekan lama ikon aplikasi Tumbuhin di homescreen.
2 Pilih "Uninstall" atau "Copot Pemasangan".
3 Konfirmasi penghapusan. Seluruh data aplikasi (transaksi, tabungan, preferensi) akan terhapus permanen dan seketika dari perangkat Anda.

Metode 2 — Hapus Data Tertentu Tanpa Uninstall

1 Buka aplikasi Tumbuhin, masuk ke menu Profil.
2 Untuk menghapus transaksi tertentu: buka menu Riwayat Transaksi, geser (swipe) transaksi yang ingin dihapus, lalu konfirmasi penghapusan.
3 Untuk menghapus seluruh data aplikasi tanpa uninstall: buka Pengaturan Android > Aplikasi > Tumbuhin > Penyimpanan > Hapus Data (Clear Data).
💡 Tips: Sebelum menghapus data, Anda dapat mengekspor seluruh riwayat transaksi terlebih dahulu melalui menu Profil > Ekspor Data sebagai cadangan pribadi dalam format CSV.

📊 Jenis Data dan Masa Retensi

Jenis Data Disimpan Di Masa Retensi
Transaksi (pemasukan/pengeluaran) Lokal di perangkat Selama aplikasi terinstal, atau sampai dihapus manual
Target & kontribusi tabungan Lokal di perangkat Selama aplikasi terinstal, atau sampai dihapus manual
Nama panggilan & preferensi Lokal di perangkat Selama aplikasi terinstal, atau sampai dihapus manual
Log kerusakan teknis (crash report) Server pihak ketiga (Sentry) Maksimal 90 hari, otomatis terhapus setelahnya

Tidak ada data pengguna yang disimpan di server milik developer Tumbuhin. Satu-satunya data yang meninggalkan perangkat pengguna adalah log teknis kerusakan aplikasi (tidak termasuk data finansial), yang dikelola oleh layanan pemantauan pihak ketiga dan dihapus otomatis dalam 90 hari.
