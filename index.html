🌳 Data yang Kami Simpan

Tumbuhin menyimpan seluruh data transaksi, tabungan, dan preferensimu secara lokal di perangkatmu sendiri. Kami tidak mengunggah, membagikan, atau menjual data keuanganmu kepada pihak ketiga mana pun.

Data yang tersimpan meliputi: catatan pemasukan dan pengeluaran, target tabungan, kategori transaksi, tag mood, catatan pribadi, serta preferensi tampilan aplikasi (seperti mode terang/gelap).

🔒 Keamanan Data

Kamu dapat mengaktifkan kunci biometrik (sidik jari atau Face ID) di menu Pengaturan untuk melindungi akses ke aplikasi. Data tetap tersimpan di perangkatmu dan tidak dapat diakses tanpa membuka aplikasi secara langsung.

Kami tidak mengoperasikan server yang menyimpan data pribadi atau finansialmu. Seluruh pemrosesan data terjadi di dalam aplikasi, di perangkatmu.

📤 Kontrol Penuh atas Data

Kamu dapat mengekspor seluruh data transaksimu kapan saja dalam format CSV melalui menu Ekspor Data di Pengaturan, dan memulihkannya kembali lewat menu Impor Data.

Menghapus aplikasi akan menghapus seluruh data secara permanen dari perangkatmu. Kami menyarankan untuk mengekspor data secara berkala sebagai cadangan pribadi.

📊 Laporan Kerusakan Teknis

Untuk membantu kami memperbaiki bug, aplikasi ini menggunakan layanan pemantauan kerusakan teknis (crash reporting) pihak ketiga. Layanan ini hanya mengirimkan informasi teknis terkait error aplikasi (seperti jenis kesalahan dan informasi perangkat), tidak termasuk data transaksi, saldo, atau catatan pribadimu.

👤 Hak Kamu

  • Kamu berhak mengakses, mengekspor, dan menghapus datamu kapan saja.
  • Kamu berhak menonaktifkan fitur apa pun yang tidak ingin digunakan.
  • Kamu tidak akan menerima iklan atau promosi dari pihak ketiga melalui aplikasi ini.

📜 Perubahan Kebijakan

Kebijakan ini dapat diperbarui seiring pengembangan aplikasi. Perubahan signifikan akan diinformasikan melalui pembaruan aplikasi atau catatan rilis di Play Store.
